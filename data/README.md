## Data

The original competition datasets (`train.csv` and `test.csv`) are not included in this repository because of size restrictions.

To run the project:

1. Create a folder named `data` in the repository root.
2. Place `train.csv` and `test.csv` inside the `data` folder.
3. Run `scripts/catboost_03062026.R`.

The script will generate `submission.csv`.
